# C++库学习笔记

个人认为对于一个C++库的学习有四重境界:

1. 了解接口
2. 了解实现
3. 理解
  理解这个库面临的问题,解决的方法.这其中又反映出了哪些问题特定,语言相关的东西.
4. 应用
  * 正确
  * 高效: 开发效率/运行效率
  * 会通以求超胜
